# mp

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
功能：登陆、注册、上传、点赞、获取推荐/我的作品列表
```
